这是一个简易的关于 spring-jdbc-tx-showcase 的演示程序，通过构建简单的项目环境整合开发和测试该框架的相关功能和特性，故而便于学习，积累和分享！

> 简介

- 基于 Spring3 + Hiberate4 + H2/Mysql 整合开发且包含基本的CRUD功能
- 基于 Maven3 构建项目
- 基于 Bootstrap3 的前端页面开发

> 运行

- git clone
- mvn jetty:run

> 注意

- 若使用MySQL数据库,请先修改用户名和密码
